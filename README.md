Riddle Alley - Tienda e-commerce

Creado por Sofía Rosalez para el proyecto de Talento Tech.

Este proyecto se basa en un case study de UX llamado Riddle Alley, una tienda de tecnología que necesitaba el diseño de una experiencia de compra online.

El objetivo de este proyecto fue practicar diseño de interfaces y la construcción de una tienda e-commerce.

Tecnologías utilizadas
HTML
CSS
Notas sobre el desarrollo

Para la creación y centrado de las tarjetas utilicé asistencia de inteligencia artificial, ya que tuve dificultades para lograr una correcta alineación y distribución con CSS.

La IA fue usada como apoyo de aprendizaje y para resolver problemas puntuales, no como reemplazo del desarrollo del proyecto.
